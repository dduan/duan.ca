What’s new in DrString 1.3

Support for Swift 5.6. Mainly, async function signatures.

What’s new in DrString 1.2

Support for macOS 11 Big Sur and Apple Silicon Macs

What’s new in DrString 1.1

New features

  • Main app now shows a preview of what result of formatting may look like.
  • Column limit can be configured with a stepper, which modifies the limit by 10.
  • Shows release notes in Help → What’s new in DrString.

What’s new in DrString 1.0

Initial release. This release includes a main app for configuring behavior of the Xcode source editor extension, along with the extension itself.
